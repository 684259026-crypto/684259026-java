# ผังงานโปรแกรมคำนวณค่าไฟฟ้า

```mermaid
graph TD
    Start([เริ่มต้น]) --> Input[รับค่าหน่วยไฟฟ้า kWh]
    Input --> Init[กำหนดค่าเริ่มต้น bill = 0, serviceFee = 50]
    
    Init --> Check1{หน่วย <= 150?}
    Check1 -- ใช่ --> Calc1[ค่าไฟ = หน่วย * 3.50]
    Check1 -- ไม่ใช่ --> Check2{หน่วย <= 400?}
    
    Check2 -- ใช่ --> Calc2[ค่าไฟ = 150*3.50 + หน่วย-150 * 4.20]
    Check2 -- ไม่ใช่ --> Calc3[ค่าไฟ = 150*3.50 + 250*4.20 + หน่วย-400 * 5.00]
    
    Calc1 --> Total[ยอดรวม = ค่าไฟ + ค่าบริการ]
    Calc2 --> Total
    Calc3 --> Total
    
    Total --> Output[แสดงรายละเอียดและยอดรวม]
    Output --> End([สิ้นสุด])
```

## รายละเอียดขั้นตอน
1. **เริ่มต้น**: รับข้อมูลจำนวนหน่วยไฟฟ้าที่ใช้ (kWh)
2. **เงื่อนไขการคำนวณ**:
    - **0 - 150 หน่วย**: คิดหน่วยละ 3.50 บาท
    - **151 - 400 หน่วย**: ส่วนที่เกิน 150 หน่วย คิดหน่วยละ 4.20 บาท
    - **401 หน่วยขึ้นไป**: ส่วนที่เกิน 400 หน่วย คิดหน่วยละ 5.00 บาท
3. **ยอดสุทธิ**: นำค่าไฟที่คำนวณได้บวกเพิ่มค่าบริการคงที่ 50 บาท
4. **แสดงผล**: พิมพ์สรุปยอดที่ต้องชำระออกทางหน้าจอ
