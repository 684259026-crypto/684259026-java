# Electricity Calculator (Java) | โปรแกรมคำนวณค่าไฟฟ้า (Java)

## English Description
A simple Java program to calculate electricity bills based on progressive consumption rates.

### Electricity Rates
*   **0 - 150 units**: 3.50 Baht/unit
*   **151 - 400 units**: 4.20 Baht/unit
*   **401 units and above**: 5.00 Baht/unit
*   **Service Fee**: 50.00 Baht (Fixed)

### How to Run
1. Compile the program:
   ```bash
   javac ElectricityCalculator.java
   ```
2. Run the program:
   ```bash
   java ElectricityCalculator
   ```

### Example Output
```
Enter electricity units consumed (kWh): 200
------------------------------------
Electricity Consumption: 200.0 kWh
Base Electricity Cost: 735.00 Baht
Service Fee: 50.0 Baht
------------------------------------
Total Amount to Pay: 785.00 Baht
------------------------------------
```
