# โปรแกรมคำนวณค่าจัดส่งสินค้า

โปรแกรมนี้เป็นระบบที่ใช้สำหรับคำนวณค่าจัดส่งสินค้าตามน้ำหนัก ประเภทการจัดส่ง และสถานะสมาชิก VIP

## คุณสมบัติ

- คำนวณค่าจัดส่งตามน้ำหนักของพัสดุ
- รองรับการจัดส่งแบบธรรมดาและแบบด่วน
- รองรับส่วนลดสำหรับสมาชิก VIP
- มีอัตราค่าจัดส่งที่แตกต่างกันระหว่างบริการธรรมดาและบริการด่วน

## เงื่อนไขการคิดค่าจัดส่ง

### อัตราค่าจัดส่งมาตรฐาน
- น้ำหนัก 0–1 กิโลกรัม = 40 บาท
- น้ำหนักมากกว่า 1–3 กิโลกรัม = 60 บาท
- น้ำหนักมากกว่า 3–5 กิโลกรัม = 80 บาท
- น้ำหนักมากกว่า 5 กิโลกรัม = 100 บาท

### อัตราค่าจัดส่งแบบด่วน
- น้ำหนัก 0–1 กิโลกรัม = 60 บาท
- น้ำหนักมากกว่า 1–3 กิโลกรัม = 90 บาท
- น้ำหนักมากกว่า 3–5 กิโลกรัม = 120 บาท
- น้ำหนักมากกว่า 5 กิโลกรัม = 150 บาท

### บริการเสริม
- หากเลือกบริการด่วน (Express) ให้เพิ่มค่าบริการ 30 บาท
- หากเป็นสมาชิก VIP ให้ลดค่าจัดส่ง 20%

## การใช้งาน

1. คอมไพล์โปรแกรม:
   ```
   javac ShippingCalculator.java
   ```

2. รันโปรแกรม:
   ```
   java ShippingCalculator
   ```

## โครงสร้างไฟล์

- `ShippingCalculator.java` - ซอร์สโค้ดหลักของโปรแกรม
- `algor-th.md` - ผังงานและรหัสเทียมในภาษาไทย
- `README.md` - คู่มือการใช้งานนี้

## ตัวอย่างผลลัพธ์

โปรแกรมจะแสดงตัวอย่างการคำนวณค่าจัดส่งสำหรับสถานการณ์ต่างๆ เช่น:
- น้ำหนักสินค้า 0.5 กก., บริการธรรมดา, ลูกค้าทั่วไป
- น้ำหนักสินค้า 2.0 กก., บริการด่วน, ลูกค้า VIP
- และสถานการณ์อื่นๆ อีกหลายกรณี
